# 03. 품질 검증 하네스 v1.2

> 글쓰기 에이전트가 draft.md 작성 후 **반드시 실행**. B등급(80점) 미만이면 재작성.

---

## 실행 순서

```
[Step 0] 완성도 게이트 — 1개라도 FAIL이면 즉시 재작업 (점수 무관)
[Step 1] 정량 항목 자동 측정
[Step 2] 절대 금지 스캔
[Step 3] 리캐치 스타일 구조 검증
[Step 4] 정성 항목 셀프리포트
[Step 5] 종합 점수 산출 + 시트 기록
```

---

## Step 0. 완성도 게이트 (PASS/FAIL)

| 항목 | PASS 조건 |
|---|---|
| 메타 표기 누출 | PUBLISH 블록에 편집지시문·하네스결과 등 NOTES 전용 내용 0건 |
| 이미지 마커 치환 | `[IMAGE: ...]` 마커 전부 실제 이미지 경로로 치환, 참조 파일 실제 존재 |
| 소제목 요약 카드 | 본문 소제목 개수 = 소제목 요약 카드 이미지 개수 |
| 강조 지정 | NOTES 블록 강조 지정 표에 `{{hl:}}` 2개 이상 기록 |
| CTA 타겟 존재 | CTA 2곳 모두 실제 URL/전화번호 또는 `[RICH:...]` 포함 |
| 소제목별 자사 서비스 연결 | 본문 소제목 4~6개 전부 자사 서비스명/기능 언급 1회 이상 |

이 게이트는 `agents/assembler.md`에서도 재확인 — FAIL이면 final.md/final.html 생성 중단.

## Step 2. 절대 금지 스캔

아래 패턴이 1건이라도 발견되면 즉시 재작성.

**메타 표기**: `[제목]:` `[소제목]` `---` `##` ` ``` ` `**` 그대로 노출

**블로거 톤**: 안녕하세요 / 여러분 / 독자분들 / 저는·제가 / 오늘은~알아보겠습니다 / 함께 살펴보시죠 / 도움이 되셨길 / 이웃추가 / 공감과 댓글

**격언체**: "숫자는 정직하다" / "결과는 같았다" / "그게 진짜 비용이다" / "원인은 단절이다"

**광고 표현**: 무료 진단·체험 / 100% 만족 / 강력 추천 / 지금 바로 클릭

## Step 4. 정성 항목 셀프리포트

모호하면 낮은 점수 (4 vs 3 → 3, 5 vs 4 → 4). AI 자기 출력에 후하게 매기지 말 것.

| 항목 | 5점 기준 | 3점 기준 | 1점 기준 |
|---|---|---|---|
| 검색 의도 부합도 | 0.5초 내 "내 문제다" 인식 | 가치 제안 모호 | 검색 상황 파악 어려움 |
| 일반론 회피도 | 모든 단락에 수치·사례 | 절반 단락에 일반론 | 누구나 쓸 수 있는 글 |
| 종결 어미 혼용 | 격식 65~69% (이상적) | 격식 86~94% | 단일 톤 100% |
| 도입 훅 자연스러움 | 이모지+공감+자연스러운 진입 | 이모지 또는 공감 한쪽 누락 | "안녕하세요" 또는 즉시 본론 |
| 구체 수치·고유명사 밀도 | 모든 단락에 수치·업종·기간 1개+ | 60% 단락에 등장 | 거의 없음 |
| 상업적 설득력 | 모든 소제목이 문제→서비스 해결 | 2개 소제목 일반론으로 끝남 | 시장 리포트처럼 읽힘 |
| 발행 가능성 | Step 0 PASS + 그대로 복붙 가능 | 3~5곳 수정 필요 | Step 0 FAIL / 재작성 권장 |

**평균 4점 미만 → 재검토 필수. Step 0 FAIL → 발행 가능성 자동 1점.**

종결 어미 4점 권장 범위: 격식 70~85%.

## Step 5. 종합 점수

```
정량 점수 = (통과 항목 / 전체) × 100
정성 점수 = (평균 / 5) × 100
종합 점수 = (정량 × 0.6) + (정성 × 0.4)
```

| 점수 | 등급 | 처리 |
|---|---|---|
| 95~100 | A+ | 검토 대기 |
| 88~94 | A | 검토 대기 |
| 80~87 | B | 검토 대기 + 보완 권장 사항 메모 |
| 70~79 | C | 마케터 사전 확인 요청 |
| 70 미만 | F | **재작성. 시트 출력 금지.** |
